Brothaman APT Repository

This is an APT repository hosted on GitHub Pages for Brothaman.

Quick Setup

# Download and run setup script
curl -fsSL https://dev.github.io/workspace/apt/repo-setup.sh | bash

# Or manual setup:
curl -fsSL https://dev.github.io/workspace/apt/apt-repo-pubkey.asc | sudo tee /etc/apt/trusted.gpg.d/workspace.asc > /dev/null
echo "deb https://dev.github.io/workspace/apt stable main" | sudo tee /etc/apt/sources.list.d/workspace.list
sudo apt update

Available Packages

brothaman-helper (1.0.0)

Network namespace helper utility for brothaman A security-focused helper utility that allows specific network commands to be executed within network namespaces owned by the same user. . This tool provides controlled access to network namespaces for utilities like curl, ip, netstat, and systemd-socket-proxyd while maintaining security by validating namespace ownership.

sudo apt install brothaman-helper

brothaman-scripts (1.0.0)

Brothaman shell script collection and shared library A comprehensive collection of shell scripts for managing containers, ZFS storage, and system services with brothaman. Includes the shared bro-common library used by all brothaman tools. . Scripts include: - bro-compose: Container compose management - bro-service: Service lifecycle management - bro-install-*: Installation utilities for dependencies - bro-user: User management utilities - bro-doctor: System diagnostics and health checks . All scripts follow consistent conventions and use the shared bro-common library for logging, version comparison, and system utilities.

sudo apt install brothaman-scripts
